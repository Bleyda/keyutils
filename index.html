These tools are used to control the key management system built into the Linux kernel.

To build and install the tools and manual pages, run:

make
make install

If your glibc does not contain definitions for the new error codes and system calls, then try:

make NO_GLIBC_KEYERR=1

The tools are licensed under the GPL and the utility library under the LGPL. Copies of these are included in this tarball.
